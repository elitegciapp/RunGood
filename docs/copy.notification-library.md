# Copy + Notification Library (v1)

## Voice
Dry realism + encouraging.
Short sentences. No hype. No guilt.

## UI Microcopy Principles
- Neutral when user skips
- Praise effort, not perfection
- Offer options, not commands

## Missed Workout Banner Examples
- “Life got in the way. Want to adjust?”
- “No problem. Pick what you want to do next.”
Buttons:
- Reduce intensity
- Skip and continue
- Reset week

## Motivational Nudges (Push Notifications)
- “Ten minutes counts.”
- “Short run beats no run.”
- “Want the easy option today?”
- “Two runs a week still works.”

## Workout Completion
- “Logged. That’s a win.”
- “Done. Keep it simple.”
- “Good work. No need to overthink it.”

## Avoid
- “You missed…”
- “Don’t break your streak…”
- “Get back on track…”
- “No excuses…”
